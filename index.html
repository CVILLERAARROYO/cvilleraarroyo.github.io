Material Design para Bootstrap

Crea proyectos adaptables de primer nivel en la web con el framework de front-end de Material Design más popular del mundo, Material Design for Bootstrap es un kit de herramientas de código abierto basado en Bootstrap para desarrollar aplicaciones de diseño de materiales con HTML, CSS y JS. Haga un prototipo rápido de sus ideas o cree su aplicación completa con nuestras variables y mixins de Sass, sistema de cuadrícula sensible, componentes precompilados extensos y potentes complementos basados en jQuery.
